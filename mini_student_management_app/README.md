## About this app

This project is a student management application built with Flutter.
It includes student listing, search, add/edit forms, and local storage using SQLite.

## Run locally

1. Open a terminal in `mini_student_management_app`.
2. Enable Windows Developer Mode if you have not already:
   - Open Settings → Privacy & security → For developers → Developer Mode.
3. Fetch dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

If `flutter pub get` fails due to Windows symlink restrictions, enabling Developer Mode usually resolves it.
